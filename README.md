
# Kroman

Kroman is ultimate Korean hangul romanization tool. It has a lot of
implementations.

This is the Objective-C library.

## Installation

Install it with CocoaPod.

## Usage

``` objc
[Kroman parse: @손목시계]; // => son-mog-si-gye
```
